# Gothic Lockpick Solver — Contributing

Contributions are welcome! This is a single-file HTML project, so getting started is straightforward.

## Getting Started

1. Fork the repository.
2. Open `gothic_lockpick.html` in your browser — no build step required.
3. Edit the file and refresh to test.

## Guidelines

- Keep everything in a single HTML file (HTML + CSS + JS).
- No external dependencies or build tools.
- Test in at least Chrome and Firefox before submitting.
- Follow the existing code style (ES5-compatible class methods, `var` declarations).

## Reporting Issues

Please open an issue on [GitHub](https://github.com/gaonmaor/gothic_lockpick_solver/issues) with:
- Steps to reproduce
- Expected vs. actual behavior
- Browser and OS information
- Puzzle JSON (use "Show JSON") if the issue is puzzle-specific
